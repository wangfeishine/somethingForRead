1.ANSI标准在结构方面最主要到变化是定义来结构的赋值操作----结构可以拷贝，赋值，传递给函数，函数也可以返回结构类型到返回值。

2.ANSI标准中，自动结构和数组现在也可以进行初始化。

3.struct point {
      int x;
      int y;
  };

4.如果结构声明的后面不带变量表，则不许要为它分配存储空间，它仅仅描述了一个结构到模板或轮廓。但是，如果结构声明中带有标记，那么在以后定义结构实例时便可以使用该标记定义。

5.结构可以嵌套

6.结构的操作：做了一个整体复制和赋值，通过&运算符取地址，访问其成员。

7.函数参数是结构类型到话，和其他参数一样到，都是通过值进行传递的。

8.struct point origin, *pp;

  pp = &origin;
  /* (*pp).x = pp->x */
  printf("origin is (%d,%d)\n", (*pp).x, (*pp).y);
